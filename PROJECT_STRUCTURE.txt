HOW TO RUN
----------
1. python setup.py          # install dependencies (run once)
2. python ml/train.py       # train the ML model
3. python run.py            # start API on http://localhost:8000
4. open frontend/rx-mortem.html in your browser

NOTES
-----
- Always run all commands from the project root (rx-mortem-v2/)
- rf_model.pkl and scaler.pkl are created by train.py — do not create manually
- model_info.json placeholder is overwritten by train.py with real metrics
- If yara-python install fails, the system uses a built-in fallback scanner
